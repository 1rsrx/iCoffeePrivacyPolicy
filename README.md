# iCoffee プライバシーポリシー

## 1. 取得するデータと目的

私は、下記データをサードパーティ製品を使用し、収集します。

- クラッシュの計測
    - ユーザー体験を向上させるため
- 外部Webページ表示の計測
    - ユーザー体験を向上させるため
- 広告に関する情報(IDFAの取得)
    - 表示する広告をカスタマイズするため
